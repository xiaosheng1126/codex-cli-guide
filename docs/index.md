# Codex CLI Guide

这是一个“开箱即用”的文档发布模板，目标是让你最快速地从 0 到 1 拥有在线文档。

## 你现在具备的能力

- 本地写 Markdown 文档
- 使用 MkDocs 统一构建站点
- push 到 `main` 后自动部署到 GitHub Pages
- 自动生成与你当前仓库一致的文档链接

## 在线地址规则

文档默认地址固定为：

`https://<用户名>.github.io/<仓库名>/`

例如 `alice/codex-cli-guide`：

`https://alice.github.io/codex-cli-guide/`
